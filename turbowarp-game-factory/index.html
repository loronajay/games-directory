WHAT IT IS
Every TurboWarp game tends to require the same setup work: controls, UI systems, text rendering, and input handling.

TurboWarp Game Factory is designed to remove that repeated setup and replace it with a reusable foundation.
→ Pre-built control systems for keyboard and gamepad support
→ Reusable modules that can carry from game to game
→ A consistent development structure instead of rebuilding from scratch
→ A workflow where the game itself becomes the main focus
CURRENT SYSTEM
TurboWarp Game Factory is already being built as a real modular system, not just an idea.
✔ Factory Controls with keyboard and gamepad support
✔ Gamepad detection integrated into the control pipeline
✔ Shared input behavior for UI navigation and gameplay structure
✔ Text renderer development in progress as part of the core system vision
✔ Modular architecture intended to eliminate repeated setup across future games
