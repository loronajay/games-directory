1. Domain Router (Entry Control)

Your root page routes users depending on the subdomain.

Examples:

jayarcade.com → main selector

play.jayarcade.com → game grid

invest.jayarcade.com → investor page

4. Game Runtime Layer

Each game is fully isolated.

TurboWarp export → wrapped in a runtime.

Example structure:

games/
   apple-catcher/
      index.html
      script.js
      assets

The runtime does:

Scratch VM boot

extension loading

cloud variable provider

framerate control

texture limits

Example:

scaffolding.height = 360;
vm.setFramerate(30);
vm.setRuntimeOptions({
  fencing: true,
  miscLimits: true
});

So every game launches inside a standardized VM wrapper.
