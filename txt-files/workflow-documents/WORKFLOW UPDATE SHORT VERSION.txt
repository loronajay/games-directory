ARCADE UPDATE CHEATSHEET

Step 1 — Export games
Export updated games from TurboWarp Packager as ZIP files.

Step 2 — Place ZIP files in exports folder

C:\Users\leoja\Desktop\Dad Games\full-games\exports

Example:

apple-catcher.zip
art-of-war.zip
bird-duty.zip
blade-and-block.zip
dodgeballs.zip
mini-arcade.zip
sumorai.zip
x-pong.zip

Step 3 — Run build script

From the arcade repo root:

python scripts/build_arcade.py

This will automatically:

• Import all ZIP files into games/
• Replace old game files
• Patch HTML with mobile controller + meta tags + GoatCounter

Step 4 — Deploy site

python scripts/build_arcade.py --push

This will:

• Import games
• Patch games
• Commit changes
• Push to GitHub

Step 5 — Clean processed ZIP files (optional)

python scripts/build_arcade.py --push --clean-exports

This will:

• Import games
• Patch games
• Push changes
• Delete processed ZIP files from exports

Step 6 — Update preview videos (manual)

Add or update videos in:

previews/

Example:

previews/apple-catcher.mp4
previews/x-pong.mp4

PATCH ONLY (if no game exports changed)

python scripts/patch_all_games.py --push

DRY RUN (test build without changing files)

python scripts/build_arcade.py --dry-run
